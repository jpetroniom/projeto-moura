# projeto-moura

Landing page estática, protótipo (fase 1 — pré-visualização privada).

Sem build step: abrir `index.html` direto no navegador ou publicar como site estático (Cloudflare Pages / Vercel / Netlify).

`robots.txt` e a meta tag `noindex` bloqueiam indexação por buscadores enquanto o conteúdo está em validação.
